Question


Write a progam to Construct binary tree using the given Array. Array represents tree traversed in dsf with null nodes as well represented as n.

Eg. Array: ["50", "25", "12", "n", "n", "37", "n", "n", "75", "62", "n", "n", "87", "n", "n"]

Tree will look like:


Input


First line contain integer n that is size of array.
Next line contain n numbers of the array, null is represented with n.

15
50 25 12 n n 37 n n 75 62 n n 87 n n


Output


25 <- 50 -> 75
12 <- 25 -> 37
. <- 12 -> .
. <- 37 -> .
62 <- 75 -> 87
. <- 62 -> .
. <- 87 -> .
